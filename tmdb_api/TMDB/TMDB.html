Creates a instance of TMDB client.

Use apiKeys to provide your TMDB apikeys, you can find yours at https://www.themoviedb.org/settings/api. Provide logConfig to configure your logs from TMDB library, by default it shows none. You can provide your own Dio instance with dio, which can be used for testing. caching and other dio intercepters with interceptors. defaultLanguage sets a default language parameter to all supported methods, by default it is set to en-US.
