Get a list of upcoming movies in theatres. This is a release type query that looks for all movies that have a release type of 2 or 3 within the specified date range.

You can optionally specify a region parameter which will narrow the search to only look for theatrical release dates within the specified country.

Parameters

movieId: Id of a given movie.

language: Pass a ISO 639-1 value to display translated data for the fields that support it. minLength: 2 pattern: (a-z{2})-(A-Z{2}) default: en-US

page:Specify which page to query. minimum: 1 maximum: 1000 default: 1

region:Specify a ISO 3166-1 code to filter release dates. Must be uppercase. pattern: ^A-Z{2}$

Future<Map> getUpcoming({
  String? language,
  int page = 1,
  String? region,
}) {
  if (page < 1 || page > 1000) {
    throw ArgumentError('movieId < 1 || page < 1 || page > 1000 is true');
  }

  final para = <String>[
    'language=${language ?? _v._tmdb.defaultLanguage}',
    'page=$page'
  ];
  if (region != null) {
    para.add('region=$region');
  }

  return _v._query('$_endPoint/upcoming', optionalQueries: para);
}
